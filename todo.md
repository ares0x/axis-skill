# 🚀 sanage Axis 数据采集与上线准备清单 (Todo.md)

本清单旨在指导采集 2026 年高考志愿推演所需的核心硬核数据。完成这些数据的采集并转化为对应的文件后，项目即可完全跑通并投入生产使用。

## 🟩 一、 静态知识库数据采集 (Priority: HIGH)
> 地点：`data/` 目录
> 目的：为 `veto_officer`（熔断官）和 `evaluator.py` 提供查杀和红利对齐的绝对依据。

### 1. 采集全国高校撤销专业名单
- [ ] **任务**：获取教育部最新公布的全国普通高等学校本科专业备案和审批结果。
- [ ] **格式**：转换为 `data/2026_cancelled_majors.csv`
- [ ] **表结构**：`major_name (专业名称), risk_level (风险等级: High/Medium), reason (撤销原因简述/AI替代率说明)`
- [ ] **重点采集对象**：2024-2026年高频被撤销的传统商科（如部分学校的大数据与会计、市场营销）、传统土建类、学前教育等。

### 2. 采集全国高校新增专业名单
- [ ] **任务**：获取教育部 2026 年最新审批通过的本科新专业（如具身智能、低空经济、海洋智能等新质生产力方向）。
- [ ] **格式**：转换为 `data/2026_added_majors.csv`
- [ ] **表结构**：`major_name (新专业名称), industry_tag (产业标签), core_skills (核心硬核工具技能)`

### 3. 提炼“十五五”产业规划核心词
- [ ] **任务**：查阅国家及各主要经济圈（长三角、珠三角、成渝等）最新的“十五五”规划建议，提取国家重点补贴、扶持的战略性新兴产业。
- [ ] **格式**：更新至 `data/15th_five_year_plan.md`
- [ ] **内容要求**：提炼出诸如“低空物流、高端半导体设备、生物医药合规、具身智能视觉”等高内驱力关键词。

## 💡 数据采集的小窍门（如何快速获取这些数据？）
1. **官方渠道**：直接去**中国教育部官网**搜索“2026年普通高等学校本科专业备案和审批结果”（通常在每年3-4月公示），这是最权威的撤销与新增专业数据源。
2. **地方考试院**：针对广东春考和艺考，直接打包下载**广东省教育考试院**发布的当年招生专业目录和投档录取办法。
3. **专家经验提炼**：参考你之前提到的 `ZhangXueFeng-skill` 等开源项目，直接把其中已经验证有效的考公、考研跨考路线以规则形式固化下来。
