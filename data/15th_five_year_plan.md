# “十五五”产业规划核心词（关键词库）

> 目的：面向专业设置/人才培养/产业研究，提炼国家及主要经济圈在“十五五”相关《规划纲要/规划建议》中反复出现、具备政策资源投入指向的**战略性新兴产业**与**未来产业**关键词。

## 1. 国家层面（《中华人民共和国国民经济和社会发展第十五个五年规划纲要（2026—2030年）》）

**来源**：国家发展改革委发布 PDF《中华人民共和国国民经济和社会发展第十五个五年规划纲要（2026—2030年）》
https://www.ndrc.gov.cn/fggz/fzzlgh/gjfzgh/202603/U020260317369114704096.pdf

### 1.1 战略性新兴产业（“发展壮大新兴产业”）

纲要明确加快发展：
- 新一代信息技术
- 新能源 / 新型储能 / 新型太阳能电池
- 新材料
- 智能网联新能源汽车 / 智能驾驶
- 机器人
- 生物医药 / 创新药临床使用
- 高端装备
- 航空航天（含国产大飞机规模化系列化）
- 海洋经济
- **低空经济（健康有序发展）**
- 北斗系统创新应用

### 1.2 未来产业（“前瞻布局未来产业”）

纲要明确前瞻布局：
- 量子科技
- **生物制造**
- 氢能与核聚变能
- **脑机接口**
- **具身智能**
- 第六代移动通信（6G）

### 1.3 “新产业新赛道”典型关键词（专栏摘录）

纲要专栏对“新赛道”给出更细颗粒度方向（可直接作为人才培养与课程模块关键词）：
- **集成电路**：先进制程能力、关键装备/材料/零部件、宽禁带/超宽禁带半导体（如氧化镓、金刚石）、存算一体、三维集成、光电融合
- **具身智能**：具身智能实训场、虚实融合协同训练、物理AI、大小脑一体化具身模型、核心零部件、本体、人形机器人落地
- **生物制造**：酶制剂、生物种质智能设计、智能发酵、生物育种/生物化工/生物医药/生物能源；细胞与基因治疗、抗体/核酸/放射性药物
- **新型电池**：高容量电极材料、高电导率电解质、复合集流体；高精度涂覆/高速叠片等高端制造装备
- **商业航天**（专栏列出为新赛道之一）


## 2. 长三角（以上海为代表）

**代表性来源（上海“十五五”规划建议）**：上海市政府官网发布《中共上海市委关于制定上海市国民经济和社会发展第十五个五年规划的建议》
https://www.shanghai.gov.cn/nw4411/20260119/4c72d40dbda44ea2bc71d384068a7edb.html

### 2.1 先导产业与“高投入”方向
- **集成电路（装备能级 + 制造 + 设计全链条突破）**
- **生物医药（创新药械研发、临床试验提效、多元支付创新等）**
- **人工智能（智算芯片/语料/智算集群协同；基础大模型迭代；具身智能技术熟化；垂类模型与多智能体开发；AI治理）**

### 2.2 新兴支柱产业（结构化表达）
- 新一代电子信息（以智能终端创新带动）
- 智能网联新能源汽车
- 高端装备
- 先进材料
- 新能源及绿色低碳
- 时尚消费品品牌化

### 2.3 未来产业“六大方向”与敏捷布局
- 未来制造、未来信息、未来材料、未来能源、未来空间、未来健康
- 重点点名：量子科技、脑机接口、可控核聚变、生物制造、6G

### 2.4 城市更新（与新专业“城市更新”强关联）
- **城市更新**、城中村改造、老旧设施更新、产业用地融合管理、“工业上楼”等


## 3. 珠三角（以广东/粤港澳大湾区为代表）

**代表性来源（广东“十五五”规划建议全文）**：广东省相关政府网站转载的《广东“十五五”规划建议全文发布》
http://www.pwccw.gd.gov.cn/szyw/content/post_1313220.html

### 3.1 数据要素与算力底座（强政策抓手）
- **数据要素基础制度**、行业高质量数据集
- 广州/深圳**数据交易所**
- 开放共享安全的数据市场
- 全国一体化算力网络**大湾区国家枢纽节点（韶关数据中心集群）**
- 覆盖全省的智能算力网络（算力-电力统筹）

### 3.2 低空经济（高频、强落地）
- **低空飞行基础设施与智能网联系统**
- 低空飞行器制造
- **城市物流**与跨珠江口常态化低空飞行（低空物流/低空通勤/低空文旅等场景）

### 3.3 “AI + 机器人”与智能终端
- “数据集+算力+大模型”体系
- **人工智能+机器人**（软硬协同、机电一体化优势）
- 智能终端产品创新

### 3.4 优势产业与关键突破
- 新能源车船
- 集成电路
- 新型储能
- 医药与医疗器械
- 无人机/无人船
- 工业母机、高端仪器、基础软件、先进材料、生物制造

### 3.5 绿色经济形态（可作为区域特色关键词）
- “绿电+储能”一体化
- 零碳工厂/零碳园区
- 水经济、林木经济、林下经济、碳汇经济

## 5. 可直接复用的“高内驱力关键词”清单（建议用于选专业/课程/研究选题）

> 说明：以下是对上述来源中“重复出现 + 可落地 + 具政策抓手”的关键词做的二次提纯（偏“能写进项目/能对接补贴/能落地场景”的表述）。

### 5.1 空天/低空经济
- 低空物流（城市物流、跨江跨海通勤）
- 低空安全管理（适航审定、空域精细化管理、运行安全保障）
- 低空飞行基础设施与智能网联系统
- 工业无人机/无人船
- 商业航天（关键能源系统、星载计算、轻量化结构等）

### 5.2 半导体与“硬科技装备”
- 高端半导体**工艺与装备**（关键设备/材料/零部件）
- 宽禁带/超宽禁带半导体（氧化镓、金刚石）
- 存算一体、三维集成、光电融合
- 工业母机（高端数控机床）、高端仪器仪表、基础/工业软件

### 5.3 人工智能与数据要素
- 数据要素市场（数据资产化、数据交易所、行业高质量数据集）
- 智算底座（算力枢纽节点、智算集群协同）
- 垂类大模型 / 多智能体（面向产业落地）
- AI治理与合规（算法治理、数据安全、隐私合规）

### 5.4 具身智能与机器人
- 具身智能实训场 / 虚实融合协同训练
- 物理AI、大小脑一体化具身模型
- 人形机器人（核心零部件、本体、场景落地）

### 5.5 生物医药与生物制造（“合规+工程化”）
- 创新药械研发与临床试验提效
- 生物制造：智能发酵、酶制剂、生物种质智能设计
- 细胞与基因治疗、抗体/核酸/放射性药物
- 生物医药合规（GMP/临床/医保支付与采购机制）

### 5.6 新能源与储能
- 新型储能（电化学储能系统集成、储能安全）
- 新型电池关键材料（电极/电解质/复合集流体）
- “绿电+储能”一体化、零碳工厂/零碳园区
- 氢能与“氢走廊”示范

### 5.7 城市更新与韧性治理
- 城市更新（存量改造、老旧设施更新、城中村改造）
- 空间数据治理（GIS/时空数据）
- 工业上楼、产业用地融合管理


## 6. 备注：如何用这份关键词库做“专业设置/选专业”

- **撤销高频传统专业**可用“关键词→课程模块”重构（如：会计学→数智财务/合规风控/数据治理；土木工程→智能建造/BIM/城市更新）。
- **新增专业**通常对应“政策赛道 + 场景牵引 + 工具链硬技能”，建议以“数据/算力/合规/工程化”作为通用底座。
